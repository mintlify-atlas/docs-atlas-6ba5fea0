
```tsx showLineNumbers

```

## Props

| Prop        | Type                                       | Default | Description                                          |
| ----------- | ------------------------------------------ | ------- | ---------------------------------------------------- |
| `children`  | `React.ReactNode`                          | `-`     | The content of the button.                           |
| `className` | `string`                                   | `-`     | Additional class names for the button.               |
| ...rest     | Extends both `MotionProps` and `React.HTMLAttributes<HTMLElement>` | `-` | Supports all motion and HTML attributes. |

## Examples

### Custom Styling

```tsx showLineNumbers
<ShinyButton className="bg-blue-500 text-white">
  Custom Shiny Button
</ShinyButton>
```

### With Motion Props

```tsx showLineNumbers
<ShinyButton 
  whileHover={{ scale: 1.05 }}
  transition={{ duration: 0.2 }}
>
  Hover to Scale
</ShinyButton>
```
