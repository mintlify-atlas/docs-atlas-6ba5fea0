## What's Included

All templates include:

- **Modern Tech Stack**: Built with Next.js, React, TypeScript, and TailwindCSS
- **Dark Mode**: Full dark mode support with next-themes
- **Responsive Design**: 100% mobile responsive layouts
- **SEO Optimized**: Full SEO coverage with dynamic OpenGraph images
- **Animations**: Smooth animations and micro-interactions
- **Clean Code**: Maintainable code with global constants and variables
- **Quick Setup**: 1-click download and 5-minute customization
- **Deploy Ready**: Deploy live to Vercel instantly

## Getting Started

1. Browse the templates and choose one that fits your needs
2. Preview the live demo to see it in action
3. Download or clone the template repository
4. Update the content, images, and colors
5. Deploy to Vercel or your preferred hosting platform

## Tech Stack

All templates are built with industry-standard technologies:

- **Next.js 14-15**: React framework for production
- **React 18-19**: Latest React features
- **TypeScript 5**: Type-safe development
- **TailwindCSS 3-4**: Utility-first CSS framework
- **Framer Motion / Motion**: Advanced animations
- **Radix UI**: Accessible component primitives
- **shadcn/ui**: Beautiful, accessible components
