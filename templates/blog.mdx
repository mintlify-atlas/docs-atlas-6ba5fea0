## Overview

A minimal changelog template to showcase your product updates, releases, and improvements in a beautiful timeline format. Built with Next.js, TailwindCSS, and Fumadocs for easy content management.

## Features

- **MDX Support**: Write posts in Markdown with JSX components
- **Timeline Format**: Beautiful chronological layout
- **Dark Mode**: Full dark mode support
- **Mobile Responsive**: Perfect reading experience on all devices
- **SEO Optimized**: Built-in SEO best practices
- **Fast Performance**: Lightning-fast page loads
- **Easy Content Management**: Simple file-based content with Fumadocs
- **Clean Design**: Minimal, distraction-free reading experience

## Tech Stack

Built with the latest technologies:

- **Next.js 15**: Latest React framework
- **React 19**: Modern React features
- **TypeScript 5**: Type-safe development
- **TailwindCSS 4**: Latest utility-first CSS
- **Fumadocs UI**: Content management
- **next-themes**: Dark mode support
- **Radix UI**: Accessible primitives
- **shadcn/ui**: Beautiful components

## Getting Started

1. Clone the repository from GitHub
2. Install dependencies with `npm install`
3. Add your blog posts in MDX format
4. Customize the theme and styling
5. Deploy to Vercel for free hosting
