## What is this?

This is a landing page template for an AI agent startup. The template is built with Next.js, TailwindCSS, and Motion, optimized for showcasing AI agent services.

## Who is this for?

You are launching a new AI agent startup and need a professional, eye-catching landing page to showcase your product's features.

## Why should I use this?

- Save 500+ hours of work
- No need to learn advanced animations
- Easy to configure and change
- 1-click download and setup
- 5 minutes to update the text and images
- Deploy live to Vercel

## Features

Comprehensive landing page with 9 sections:

### Page Sections

- **Header Section**: Clean navigation with CTA
- **Hero Section**: Compelling introduction to your AI agent
- **Logos Section**: Build credibility with brand logos
- **Features Bento Grid Section**: Showcase features in modern grid layout
- **Testimonials Section**: Social proof from happy customers
- **Pricing Section**: Clear, transparent pricing
- **FAQ Section**: Answer common questions
- **CTA Section**: Drive conversions
- **Footer Section**: Complete footer with links

### Additional Features

- **Dark Mode**: Full dark mode support
- **Mobile Responsive**: 100% mobile responsive design
- **SEO Optimized**: Full SEO coverage
- **OpenGraph Images**: Dynamic OpenGraph images for every page
- **Smooth Animations**: Tons of animations and micro-interactions
- **Clean Code**: Maintainable, well-structured code
- **Global Config**: Easy customization with global constants
